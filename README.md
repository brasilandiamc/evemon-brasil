# Evemon Brasil - Site para Vercel

Estrutura em páginas individuais:

- `index.html` → Início
- `shop.html` → Loja
- `como-jogar.html` → Como Jogar
- `downloads.html` → Downloads
- `equipe.html` → Equipe
- `assets/styles.css` → CSS principal
- `assets/shop.js` → Loja, filtros e carrinho
- `assets/common.js` → Copiar IP

## Como atualizar no GitHub

1. Extraia este ZIP.
2. Entre no repositório `evemon-brasil`.
3. Clique em **Add file > Upload files**.
4. Arraste todos os arquivos extraídos.
5. Marque para substituir os arquivos antigos.
6. Clique em **Commit changes**.

A Vercel atualiza automaticamente depois do commit.
